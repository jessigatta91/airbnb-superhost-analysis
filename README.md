## üß© Project 1 ‚Äì Tableau Interactive Storyboard

**Purpose:** Visualize and communicate key insights about factors influencing Superhost status using dashboards.

**Key Questions Addressed (Who / What / When / Where / Why):**  
- **Who:** Airbnb hosts and analysts interested in understanding Superhost dynamics.  
- **What:** Analysis of how host and listing characteristics influence Superhost status.  
- **When:** Dataset spans 2022‚Äì2024.  
- **Where:** Italy, with city-level focus including Florence, Milan, Naples, Rome, and Venice.

## üêç Project 2 ‚Äì Python Case Study

**Purpose:** Conduct in-depth data exploration and analysis of Airbnb listings to understand patterns in ratings, reviews, pricing, and host behavior. This analysis serves as a portfolio project to showcase Python and data analytics skills.  

**Structure (Beginning, Middle, End):**  
- **Beginning:** Exploratory data analysis, descriptive statistics, and correlation matrices to understand relationships between numeric features.  
- **Middle:** Advanced visualizations including:  
  - Scatterplots (Rating vs Cleaning Score, Price vs Max Guests)    
  - Clustering analysis with K-Means (Rating vs Host Experience, Price vs Rating, Bedrooms vs Rating)  
  - Geospatial analysis (City markers with average ratings, density heatmaps, combined price + density map)  
- **End:** Summary of insights and conclusions from clustering and geospatial analysis.  

**Portfolio PPT Export:** See `Portfolio/` folder.  

**Python Scripts and Notebooks:** See `Python/` folder.
